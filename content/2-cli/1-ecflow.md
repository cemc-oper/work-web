使用预定义的命令启动和停止业务账户的ecFlow服务。

### 启动ecFlow服务

使用业务账户登录指定节点（login01或login02）使用如下命令启动该节点上ECFLOW服务，以nwp户头为例：

```bash
/g1/u/nwp/bin/start_ecflow
```

### 停止ecFlow服务

使用业务账户登录指定节点（login01或login02）使用如下命令停止该节点上的ECFLOW服务，以nwp户头为例：

```bash
/g1/u/nwp/bin/stop_ecflow
```
